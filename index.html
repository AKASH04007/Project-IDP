The Smart IoT-Based Industrial Equipment Monitoring System is designed to monitor important machine parameters such as temperature, vibration, and current in real time using an ESP32 microcontroller. The system collects data from sensors and displays it on an OLED screen while also hosting a web dashboard that allows users to view the machine status through a browser over WiFi. By continuously tracking equipment conditions, the system helps detect abnormal behavior early and supports better maintenance and operational safety.
